# System Architecture

## Recommended Initial Stack
- Frontend: Next.js
- Backend: FastAPI
- Database: PostgreSQL
- Vector retrieval: pgvector
- Storage: private object storage
- Auth: Google Workspace SSO
- Renderer: Google Sheet / Docs compatible adapter

## Main Components
- Web App
- API Layer
- Operational DB
- Private Object Storage
- Knowledge / Retrieval Layer
- AI Gateway
- Rules / Policy Engine
- Quote Renderer
- Audit / Governance Layer
